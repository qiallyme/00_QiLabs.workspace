START HERE — EmpowerQNow713 Interactive Migrator V2

WHY THIS V2 EXISTS

Your dry run showed many destination files already exist. The first tool protected you by creating __migrated duplicate names.

That was safe, but messy.

This V2 is cleaner:
- If the destination file already exists, it skips it.
- It does NOT create __migrated duplicate copies by default.
- It still never overwrites anything.
- It still starts with Dry Run.
